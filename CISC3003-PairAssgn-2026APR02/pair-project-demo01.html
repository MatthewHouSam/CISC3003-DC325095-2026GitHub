Project Demo 01

TechStore E‑Commerce Landing Page

This project demo is based on the existing file cisc3003-SugEx05-demo.html. It demonstrates how our studio can build a modern e‑commerce homepage with hero banner, category cards, featured products, and call‑to‑action buttons.

Overview

Project Description

TechStore is a static multi‑page demo that simulates an online technology shop. The layout includes a navigation bar, full‑width hero section, responsive product cards, and links to additional pages such as allProducts.html and productDetails.html.

Multi‑page Responsive E‑Commerce Layout

The design applies a mobile‑first approach: on smaller screens, cards stack vertically; on wider screens, flexbox / grid arrange them into columns, similar to the example layouts shown in the pair assignment document.

Key Features

What This Demo Shows

  • Navigation bar that links to different sections and pages.
  • Hero banner with headline, subtitle, and primary call‑to‑action button.
  • Category cards with overlay effects for laptops, phones, and accessories.
  • Featured product section with images, names, prices, and buttons.
  • Footer with additional navigation and branding.

Together, these components show our ability to structure an e‑commerce front page that could later be connected to real data or back‑end services.
