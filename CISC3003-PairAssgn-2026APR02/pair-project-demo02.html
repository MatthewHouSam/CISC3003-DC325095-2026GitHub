Project Demo 02

One‑Page Personal Portfolio Layout

This demo represents the type of personal portfolio layout inspired by the Suggested Exercise 03 and the Coder Coder tutorial series. It shows how we structure a one‑page site with hero, about, experience, projects, and contact sections.

Overview

Project Description

The portfolio layout is a multi‑section single page that could represent either Lindsay or Matthew. It includes a hero banner with name and role, an about section, experience cards, project previews, and a contact area.

One‑page Portfolio Responsive Layout

In our actual assignment, this concept is implemented inside public_Lindsay/cisc3003-IndAssgn.html and public_Matthew/cisc3003-IndAssgn.html, each with its own design flavour.

Key Sections

What This Layout Includes

  • A hero section with name, role, and call‑to‑action buttons.
  • An about section describing background and learning goals.
  • Experience or skills cards summarising technical abilities.
  • A projects grid linking to detailed project pages.
  • A contact section with email and other links.

These elements form the basic structure of our individual portfolio sites and demonstrate our understanding of typical personal branding pages.
