Pair Profile

Our values, beliefs and long-term goals as a studio.

This page follows the one-page layout style suggested in the assignment instructions. It summarises our shared philosophy, career aspirations, personal interests, and a short blog-style reflection on our CISC3003 learning journey.

Studio Values

Our Values & Beliefs

Principles that guide how Lindsay & Matthew Web Studio approaches projects and collaboration.

Quality

Learning by building real layouts

We believe that the best way to learn web programming is to ship complete layouts that can be opened and tested in real browsers. Every page in this business site is designed as a polished one-page section with clear hierarchy and responsive behaviour.

Clarity

Readable content and code

We keep our HTML and CSS organised, use meaningful class names, and document our work so that instructors, teaching assistants, and potential employers can easily follow our logic.

Respect

Supporting each partner

Each partner maintains their own portfolio site, while the business site connects both of them. We respect each other's design decisions and use this assignment as a chance to learn from different design styles.

Career Aspirations

Where We Are Heading

How this course and assignment support our long-term development.

Short-Term

Solid web programming foundation

In the short term, we aim to master semantic HTML, modern CSS layout techniques, and basic JavaScript for interactivity. Completing this pair assignment is an opportunity to organise our previous work and package it as a professional-looking business website.

  • Complete all required business pages and portfolio links.
  • Apply mobile-first responsive design with flexbox.
  • Practice basic client-style communication using clear text sections.
Long-Term

Career and further study

In the longer term, we plan to use the skills from CISC3003 to support job applications, internships, and possible postgraduate studies. This studio site can be easily extended with more projects and case studies in the future.

  • Build a portfolio that can be shared with employers.
  • Use the site as a template for real freelance work.
  • Keep improving accessibility and performance.

Our Blog Posts

Reflections from CISC3003

Short reflections that could eventually be turned into real blog posts in our personal sites.

Post 01

From static pages to responsive layouts

We started with simple fixed-width pages and gradually adopted mobile-first media queries. The pair assignment forces us to think about how the same content should adapt to mobile, tablet, and desktop screens, similar to the example layouts shown in the course materials.

Post 02

Reusing individual projects in a business context

Instead of building everything from zero, we reuse existing individual assignments and suggested exercises as project demos. This reflects how real studios build case studies from previous work.
