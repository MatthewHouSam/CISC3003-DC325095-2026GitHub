Project 1: Responsive Portfolio Homepage

A one-page style responsive homepage that introduces my profile and highlights my skills as a CISC3003 student.

Project Overview

This project is the main homepage of my individual portfolio website for the CISC3003 Web Programming course. It follows the structure of a modern, responsive one-page portfolio with sections such as About Me, Experience, Projects, and Contact.

The layout is adapted from the provided tutorial template and customised with my own information, including my name, student ID, and learning background.

Technologies Used

  • Semantic HTML5 structure
  • External CSS for layout and styling
  • Responsive design with media queries
  • Simple JavaScript navigation toggle for mobile menu

Key Features

  • A hero/profile section with my own profile photo.
  • Clear navigation linking to the main sections of the homepage.
  • An experience section that summarises my technical skills and course activities.
  • A projects section linking to detailed demo pages.
  • A simple contact section with my UM email.

My Learning Reflection

While completing this project, I practised separating structure (HTML) from presentation (CSS), and I became more familiar with responsive layout techniques. I also learned how to adapt a template to my own needs while keeping the design clean and easy to read.
