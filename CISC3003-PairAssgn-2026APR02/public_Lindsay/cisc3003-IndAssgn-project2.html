Project 2: Web Programming Showcase Page

A simple showcase page summarising selected exercises and practice work from the CISC3003 Web Programming course.

Project Description

This project page acts as a demo to present some of the work I did while learning web programming. The goal is to demonstrate my ability to structure content clearly and present information in a clean, readable layout.

Showcased Items

  • Basic HTML page structure and headings.
  • Using lists, links, and images to organise information.
  • Applying external CSS to control typography and spacing.
  • Creating simple page sections that follow a one-page style.

Design Choices

The design of this page reuses the clean and modern style of the homepage, but focuses more on text content. Each section is presented in a card-like layout to make the information easier to scan and understand.

Skills Practised

  • Planning a simple information hierarchy for a web page.
  • Maintaining consistent styling across multiple pages.
  • Writing short, clear English descriptions for a technical topic.
